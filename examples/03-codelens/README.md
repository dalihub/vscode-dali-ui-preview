# 03 · CodeLens (Code-to-Preview)

For any function that returns a DALi view type, a **`▶ Preview`** CodeLens
appears just above its signature. Click it to preview *that specific function*
— handy when a file has several UI builders and you want to flip between them
without editing markers.

## Try it

1. Open [`functions.cpp`](functions.cpp).
2. A `▶ Preview` lens shows above each view-returning function.
3. Click the lens above different functions — each renders in the panel.

No save needed — the click itself triggers the preview.
